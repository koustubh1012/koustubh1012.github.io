About me

I’m a Robotics Engineer with a Master’s in Robotics from the University of Maryland, where I specialize in building intelligent, real-world deployable robotic systems. My expertise lies at the intersection of autonomous navigation, perception pipelines, and system integration for mobile and aerial robots.

My journey began with a B.Tech in Electrical Engineering from IET Lucknow, and evolved through hands-on experience with Robot Operating System (ROS), PX4 Autopilot, and embedded systems like VOXL2. I’ve developed UAV-based 3D reconstruction pipelines using ROS2, PX4, and feature-matching techniques, and deployed ground robots equipped with LiDAR, YOLOv8, and behavior planners for real-time decision-making. I've also containerized robotics pipelines with Docker and scaled them on AWS for cloud-based applications.

Prior to graduate school, I worked at Tata Consultancy Services (TCS) as an Assistant System Engineer, maintaining Salesforce Commerce Cloud storefronts for a leading e-commerce client. I thrive in fast-paced environments, enjoy attending hackathons and robotics conferences, and am always eager to collaborate on cutting-edge projects in autonomy, AI, or robot perception.

Projects

Turtlebot Perception Challenge

Designed and implemented a ROS2-based navigation pipeline for TurtleBot3 using visual cues. The system leveraged Canny edge and Hough line transforms for horizon detection, optical flow for dynamic obstacle avoidance, and centroid-based line following for path tracking. Integrated YOLOv8 for stop sign detection and executed conditional stop behavior. Validated the full stack in both Gazebo simulation and real-world environments.

Optimized 3D Reconstruction using MASt3R and UAV-Based Visual-Telemetry Fusion

Designed a real-time image acquisition system that intelligently selects frames based on feature similarity and PX4 telemetry, reducing redundant captures by 60%. Integrated VOXL SDK for onboard streaming and odometry, and used MASt3R for rapid 3D model generation (2 mins vs. COLMAP’s 30+ mins). Deployed the containerized pipeline across VOXL2 and AWS EC2, enabling scalable, high-fidelity reconstruction for indoor mapping and autonomous exploration.

Genetic Algorithm Based Path Planner

This project explores path planning using Genetic Algorithms (GA) to optimize navigation in complex environments. By leveraging evolutionary principles such as selection, crossover, and mutation, the algorithm finds efficient paths while avoiding obstacles.

Test Driven development and CI/CD for robotics using ROS2

A ROS2 simulation based project demostrating important software development practices such as Test Driven Development, Continuous Integration and Continuous Deployment, etc for ROS2 based projects

Predicting Autism Spectrum Disorder (ASD) using Eye Tracking Data and Deep Learning

This project uses deep learning to predict Autism Spectrum Disorder (ASD) from eye-tracking data. We apply CNNs, LSTMs, and Transformers to analyze the data and improve classification accuracy, aggregating multiple eye-tracking images per individual with sequence modeling to enhance diagnostic reliability.
